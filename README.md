## 📌 Descripción

Campus Santinno es un sitio web educativo desarrollado con WordPress.

El proyecto simula una plataforma donde un profesor ofrece clases
particulares y cursos, mostrando información sobre su experiencia,
metodología de enseñanza y servicios.

Este proyecto fue desarrollado como práctica para mejorar habilidades
en:

-   Desarrollo web
-   Personalización de temas de WordPress
-   Organización de proyectos
-   Uso de Git y GitHub

También forma parte de mi portafolio como estudiante de programación.

------------------------------------------------------------------------

## ⚙️ Funcionalidades

El sitio incluye:

-   Página principal informativa
-   Sección **Acerca de mí**
-   Información sobre cursos
-   Personalización mediante **tema hijo**
-   Modificaciones de estilos CSS

## 📂 Estructura del proyecto

Las modificaciones principales se realizaron dentro del **tema hijo**:

    wp-content/
     └ themes/
        └ hestia-hijo/
           ├ style.css
           └ functions.php

El uso de un **tema hijo** permite modificar el diseño y comportamiento
del sitio sin alterar el tema original.

## 🚀 Instalación del proyecto

### 1. Clonar el repositorio

``` bash
git clone https://github.com/BenjaminP-H/campus-santinno.git
```

### 2. Mover el proyecto

Colocar la carpeta dentro de:

    xampp/htdocs/

### 3. Crear base de datos

-   Abrir **phpMyAdmin**
-   Crear una base de datos nueva
-   Importar el archivo SQL si el proyecto lo requiere

### 4. Configurar WordPress

Editar el archivo:

    wp-config.php

y colocar los datos de conexión de la base de datos.

### 5. Abrir el proyecto

En el navegador:

    http://localhost/campus-santinno
